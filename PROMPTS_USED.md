# Prompts Used

Example prompts I have used during development:

* Explain a step-by-step plan to build this app from scratch.
* Suggest tools for extracting readable content from web links.
* Suggest colors that match a clean research theme.
* Improve UI layout and spacing for better usability.
* Make the app structure cleaner and easier to maintain.
* Prevent app crashes when API returns unexpected data.
* Make the website responsive.
* Check if any required features are missing.
* Add comments for code readability.
* Review the project for production readiness.
* Create .gitignore file.
* Help write basic documentation.
